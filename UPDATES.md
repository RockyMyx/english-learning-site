## 🚀 如何测试新功能

### 测试积分系统
1. 进入任何测试模式（看英文选中文、看中文选英文等）
2. 答对题目，观察首页"今日积分"是否增加
3. 刷新页面，积分应该保持（存储在LocalStorage）

### 测试访问时间
1. 打开网站，观察"今日已学习"时间
2. 等待1分钟，时间会自动更新
3. 关闭浏览器后重新打开，时间会累计（如果在1小时内）

### 测试音频播放
1. 点击任何音频按钮（🔊）
2. 应该能听到英语发音
3. 打开浏览器控制台可以看到调试信息
4. 如果播放失败，会显示错误提示

## 🛠️ 技术细节

### 积分系统
```javascript
// 答对题目时自动加1分
if (isCorrect) {
    this.score++;
    if (window.router && window.router.addPoints) {
        window.router.addPoints(1);
    }
}
```

### 访问时间计时
```javascript
// 每分钟自动更新
setInterval(() => {
    visitTime++;
    localStorage.setItem('visitTime', visitTime.toString());
    updateVisitTimeDisplay(visitTime);
}, 60000);
```
